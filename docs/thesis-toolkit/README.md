# Thesis Writing Toolkit

Bu klasör, "Yapay Zeka Destekli Nükleer Veri Analizi" tez çalışmasının
yazım sürecinde kullanılan dokümantasyon araçlarını içerir.

## İçerik

- `00-MASTER-PROMPT.md` — Tez asistanı master prompt
- `01-WORKFLOW-PLAN.md` — Faz analiz akışı
- `02-PROJECT-CLAUDE.md` — Proje kontrol dosyası
- `03-PHASE-DOC-TEMPLATE.md` — Faz dokümantasyon şablonu
- `04-THESIS-WRITING-GUIDE.md` — Akademik yazım rehberi
- `05-CODE-INSPECTION-CHECKLIST.md` — Kod inceleme protokolü
- `06-FIGURE-TABLE-CATALOG.md` — Şekil/tablo/formül master kaydı
- `07-GLOSSARY-SYMBOLS.md` — Kısaltma ve sembol sözlüğü
- `CLAUDE.md` — Skill guide (kod tabanı kuralları)
- `claude-hatalarim-ve-dersler.md` — Önceki hata kayıtları
- `tez-yazim-not-defteri.md` — Tez yazım notları
- `pipeline-hatalari.md` — Pipeline hata kayıtları
- `pc_error.md` — Donanım/PC hata kayıtları
- `faz-degerlendirme-notlari.md` — Faz değerlendirme notları

## Klasörler

- `phases/` — Her faz için ayrı analiz dokümanı
- `reports/` — Faz analiz raporları
- `thesis/` — Tez bölüm taslakları
- `sprints/` — Sprint bazlı görev takibi
- `research/` — Araştırma notları
